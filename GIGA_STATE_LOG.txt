1. OBECNY STAN SYSTEMU I ARCHITEKTURA (FAST API + NODE.JS + POSTGRESQL):
--------------------------------------------------------------------------------
System został zaktualizowany do wariantu wielo-regionalnego. Porzuciliśmy stare
zapytania i ulepszyliśmy sztuczną inteligencję XGBoost do analizy TheSportsDB
oraz Balldontlie. Mamy 3 główne serwisy:
- `data-service` (Node): pobiera the odds api
- `ai-service` (Python/FastAPI): trenuje i serwuje predykcje XGBoost (Giga Edge)
- `analysis-service` (Discord Bot): łączy dane z AI, wysyła i filtruje.

Aplikacja znajduje się w Dockerze z `docker-compose.yml` (VPS Giga Cloud).
